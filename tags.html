Important HTML Tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1–h6 headings including a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
th table header make a table header
td table data include a table data cell
div div element divide sections of the page
header header element like a div but... a header
ul ul unordered list, tag for their creation
ol ol same as above

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong strong make bold text <strong>over a million<strong> over a million
a anchor add links to page <a href="whatever.html">whatever>
img image add images to page <img src="whatever.jpg">
code code add code to the page <code>a = "super".length<code>
span span inline element for denoting a specific piece of text
li li list item, inline element within the ol, ul tags
